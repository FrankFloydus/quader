# Quader Review Artifacts

Architect review files are written here during the active architect-led workflow.

Use Windows-safe timestamps: `MM_DD_YYYY_HHMM`.

File names:

- architect review: `{task_id}_{architect-name}_review_{MM_DD_YYYY_HHMM}.md`
- root final review: `{task_id}-final-review_{MM_DD_YYYY_HHMM}.md`

Each architect review must include either `Decision: approved` or `Decision: changes-requested`.

Root archives completed or superseded review artifacts into `agents/archive/`.
