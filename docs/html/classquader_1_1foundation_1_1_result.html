Detailed Description

template<class T, class E>
class quader::foundation::Result< T, E >

Tagged success-or-error result for operations with a value payload.

Result stores either a T success value or an E error value. Calling value() or error() for the inactive state is a programmer error and is checked with QUADER_ASSERT.
